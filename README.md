The **Task Tracking App** is a RESTful backend application built using **Spring Boot** and **Spring Data JPA** that allows users to manage their daily tasks with full **CRUD (Create, Read, Update, Delete)** operations.

This project also uses **Dockerized PostgreSQL** as the database for easy setup and portability.

---

## Features

- Create a new task
- View all tasks
- View task by ID
- Update task details/status
- Delete a task
- REST API built with Spring Boot
- Database integration using JPA/Hibernate
- Docker PostgreSQL setup
- Clean layered architecture

---

##  Tech Stack

- Java 17+
- Spring Boot
- Spring Data JPA (Hibernate)
- PostgreSQL
- Maven
- Docker & Docker Compose
- Postman (API testing)
